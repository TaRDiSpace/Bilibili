## 使用方法
> 电脑打开B站网页端的视频播放页面，选择右方屏蔽设置，右键屏蔽词区域，选择导入XML文件 tv.bilibili.player.xml导入同步即可  
> 手机版需要将文件移动到 Android--data--tv.danmaku.bili--files--block-list 内，并替换文件名及文件内容中的 “替换ID” 为自己原来文件的ID
